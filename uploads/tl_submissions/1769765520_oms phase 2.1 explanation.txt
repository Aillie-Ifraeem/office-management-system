this was the core JS code i have worked on in phase 2.1:

<script>

    document.getElementById("loginForm").addEventListener("submit", async function(e) {
        e.preventDefault();
        const email = document.querySelector("input[name='email']").value;
        const password = document.querySelector("input[name='password']").value;
    
        const response = await fetch("login_process.php", {
            method: "POST",
            headers:{
                "Content-Type": "application/json"
            },
            body: JSON.stringify({email, password})
        });

        const result = await response.json();

        if(result.status == "success"){
            window.location.href = result.redirect;
        }else{
            document.getElementById("errorMsg").innerText = result.message;
        }
    });
    </script>


explanation:

firstly this code use DOM.

uses Event Listeners with async function

async function(e) {
This is the function that runs when the form is submitted

e (event object)

What normally happens?

What this line does:
Stops the default form submission
Prevents page reload

What is AJAX? (one line)

AJAX lets a webpage talk to the server and get data without refreshing the whole page.

Important Note 🧠
AJAX is not a language
AJAX is a technique

Uses:
JavaScript
API
JSON

What is an API? (in one line)
API is a messenger that lets one software talk to another software in a fixed, safe way.

Real-Life Analogy 🍽️ (Restaurant Example)
You = Customer
App = Kitchen
API = Waiter

How it works in real life:
You don’t go into the kitchen yourself ❌
You tell the waiter what you want

The waiter:
Takes your order
Goes to the kitchen
Brings back the food
➡️ You never see how food is cooked
➡️ You only care that you get the food



What is JSON? (one line)
JSON is a simple way to write and send data so computers can understand it easily.

Real-Life Analogy 📦 (Form / Packet)
Imagine you want to send details to someone.
You write it like this on paper:
Name: Ali
Age: 21
City: Karachi

That’s structured data.
👉 JSON is exactly this, but for computers.

Key Rules of JSON 🧠
✔ Data is in key : value pairs

Normal:
Ali,21,Karachi

JSON:
{
  "name": "Ali",
  "age": 21,
  "city": "Karachi"
}


✔ Keys are always in double quotes
✔ Uses {} for objects
✔ Uses [] for lists


then we also used fetch() function
The fetch() method in JavaScript is a modern, promise-based API for making asynchronous
network requests, such as fetching data from an API or submitting data to a server. 

fetch(url, options)
url → where to send
options → how to send

inside the option we have 3 parts:
